## 登录方式

### 手机号登录
1. 选择「手机号登录」
2. 输入上表手机号
3. 点击获取验证码（开发环境自动为 `123456`）
4. 输入密码与验证码
5. 勾选协议后登录

### 账号密码登录
1. 选择「账号登录」
2. 输入登录账号与密码
3. 勾选协议后登录

## 导入命令

```bash
cd Backend
npm run seed-accounts
```

全新初始化数据库时会自动包含以上账号：

```bash
npm run init-db
```

新增账号后，将 `users` 表中的用户同步到自习室：

```bash
npm run sync-study-room
```

导入测试账号时也会自动同步自习室在线用户。
