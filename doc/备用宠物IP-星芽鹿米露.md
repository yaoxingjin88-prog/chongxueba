# 备用宠物 IP：星芽鹿·米露

## 1. 角色定位

- **名称**：米露（Milu）
- **物种**：由“星种蛋”孵化的云端星芽鹿
- **核心气质**：治愈、好奇、温柔、自律
- **角色寓意**：学习像培育一颗星种，专注与坚持会让枝芽最终长成星树
- **适用方向**：成长页主宠、活动限定宠、换肤系统、3D 桌宠、Live2D 备用角色

## 2. 视觉识别基因

四个阶段都必须保留以下特征，避免进化后像不同角色：

- 薄荷青主色毛发，四肢末端渐变为淡紫色
- 额头中央的奶油白月牙印记
- 青绿色宝石大眼睛
- 珍珠白云卷胸毛与云卷尾巴
- 暖金色树枝/叶片鹿角
- 柔软长耳，耳朵内侧为淡紫色

建议主色：

| 用途 | 色值 |
| --- | --- |
| 星雾薄荷 | `#B9EEEE` |
| 月光白 | `#FFF8EC` |
| 云梦紫 | `#C8A7F2` |
| 星枝金 | `#DDB66A` |
| 宝石青 | `#167C88` |
| 夜空靛 | `#6966C8` |

## 3. 四个成长阶段

### 阶段一：星种蛋

- 云朵巢中的珍珠蛋，蛋壳下半部呈薄荷青到淡紫渐变
- 月牙、星点和云卷浮雕提前暗示角色身份
- 顶部露出两枚金色叶角幼芽
- 推荐解锁：首次进入成长系统
- 推荐动作：轻微呼吸、左右晃动、蛋壳透光、偶尔传出星点

### 阶段二：幼年体

- 大头短腿、坐姿为主，鹿角只有一片叶芽
- 性格偏好奇和依赖，适合承担陪伴与新手引导
- 推荐解锁：连续学习 3 天或累计专注 120 分钟
- 推荐动作：眨眼、抖耳、尾巴云团弹动、追逐光点

### 阶段三：成长期

- 四肢拉长，头身比下降，鹿角出现分枝与多片叶芽
- 开始掌握“聚星”能力，可抬起前蹄召唤专注星球
- 性格由依赖转为主动鼓励用户
- 推荐解锁：等级 15 或累计专注 1,000 分钟
- 推荐动作：小跑、抬蹄聚星、失败后重新集中、完成任务后跃起

### 阶段四：成熟体

- 鹿角成长为金色星树，左右各悬挂一颗星灯
- 云卷胸毛更加丰厚，增加夜空紫短披肩和星形吊坠
- 保留可爱面相，同时强化守护者与成就感
- 推荐解锁：等级 30、连续学习 30 天或完成核心成长任务
- 推荐动作：星灯摇摆、月牙光环展开、守护姿态、结算页祝福动画

## 4. 成长页呈现建议

- 宠物主视觉保持在页面上半区，四阶段共用同一锚点和视觉重心
- 切换阶段时使用“星尘聚拢 → 轮廓发光 → 新形态落地”的 600–900ms 动画
- 蛋阶段显示“孵化进度”，幼年后改为“成长经验”
- 心情、饱腹、专注三项数值可分别驱动耳朵、尾巴和星灯反馈
- 素材已按透明 PNG 输出，可直接叠加在当前星空/云海背景上

## 5. 3D 制作规格建议

- **建模风格**：圆润低中模 + 毛绒法线，不建议逐根毛发
- **面数**：移动端单角色约 12k–20k 三角面；鹿角与云卷尾优先保留轮廓
- **贴图**：主体 2048×2048，配件 1024×1024；Base Color、Normal、ORM 三套即可
- **骨骼**：约 28–36 根，耳朵、尾巴云团、鹿角吊灯需要独立辅助骨
- **表情**：眨眼、微笑、委屈、惊喜、困倦、专注 6 组 BlendShape
- **基础动画**：Idle、Blink、Happy、Study、Sleep、Feed、Evolve、Celebrate
- **阶段复用**：四阶段共用材质色板；幼年、成长、成熟可共用动画状态机，但需要独立骨架比例
